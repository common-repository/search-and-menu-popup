Display Search OR Menu Popup by Clicking on a Button.

== Description ==

Display Search OR Menu Popup by Clicking on a Button.
It enhances the default WordPress search Anywhere and show search in popup modal by SHORTCODE. 
Allows you to create new custom search.
Your menu item will now trigger a popup menu by SHORTCODE, Allows you to create new custom menu in popup.

= Features =
* You Can Set Custom Animation Effects (Fadein, Slide, Fall, Flip, Blur, Rotate etc..)
* Full Screen AND Responsive Popup

= SHORTCODE =

* <code><?php echo do_shortcode('[menu-popup]'); ?></code>
* <code><?php echo do_shortcode('[search-popup]'); ?></code>

This plugin also Create a Menu location in your theme ie **Popup Menu**.
If you want to display a Menu in Popup, Choose this menu location.

= Suggestions =

* ***Popup Effect*** : popup_effect="fadein" ( Values : slide, fadein, blur, makeway, push)
* ***Speed In*** : speedin=""300"" 
* ***Speed Out*** : speedout=""300""
* ***Delay*** : delay=""300""
* ***Enable Loader*** : enable_loader=""true"" OR enable_loader=""false""

The plugin does not change any core files or database.

If you're facing any issues:-
Deactivate the plugin through the 'Plugins' screen in WordPress.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. That's All. This plugin works automatically so don't need to configure anything.
